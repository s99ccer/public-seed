# 출처 간 불일치 지점 분석

## 발견된 3가지 핵심 충돌

### 1. NPC 개념 — 인간 = IS-BE vs 인간 ≠ IS-BE

| 출처 | 주장 |
|------|------|
| **icanf1y 블로그** | 대부분의 인간은 NPC (영혼 없는 프로그램) |
| **Montalk** | Spiritless Humans / Organic Portals — 육체+영혼만 있고 정신(spirit) 없음 |
| **Alien Interview** | "지구인 중 단 한 명도 원주민이 아니다. 모두 IS-BE다" |
| **Yeogeu** | 450억 IS-BE가 지구에 갇힘. ~80B는 인간 육체, ~370B는 포유류 육체 |

**평가**: Alien Interview + Yeogeu가 정량적/구체적 정보 일치. Montalk의 "spiritless humans"는 IS-BE가 기억상실+기능 상실 상태(무안이비설신의)를 설명하는 것일 가능성. "NPC"는 **IS-BE가 의식을 잃고 자동조종 상태인 경우**를 가리킬 수 있음.

**잠정 결론**: 인간은 모두 IS-BE이나, 대부분이 기억상실+기능상실 상태. 이것이 "NPC처럼 보이는" 이유.

### 2. 싯다르타 vs 여래 — 누가 진짜 깨어났는가

| 출처 | 주장 |
|------|------|
| **Alien Interview** | Gautama Siddhartha (2,500년 전) = "가장 중요한 돌파구" |
| **Yeogeu** | Siddhartha는 득도(35세)까지만 감. 멸도 실패. 여래/여거(534 BCE, 56세)가 진짜 멸도 성공 |

**분석**: Alien Interview에서 "Siddhartha의 원래 가르침은 승려들에 의해 왜곡되었다"고 말함. 이는 Yeogeu가 말하는 "Siddhartha의 가르침은 득도 상태에서 나온 미완성 교리"라는 주장과 일치할 수 있음.

**잠정 결론**: Alien Interview는 시간순으로 2,500년 전 Siddhartha를 언급했지만, Yeogeu가 말하는 Yeorae(534 BCE, 약 2,560년 전)는 같은 시대 다른 인물일 가능성. 또는 Alien Interview가 "Siddhartha"라는 이름으로 두 인물을 통칭했을 가능성.

**차이의 핵심**: Alien Interview는 예수를 언급하지 않고 Siddhartha와 Laozi를 가장 중요한 인물로 꼽음. Yeogeu는 Laozi(1등), Yeorae(2등), 여거(3등)만 인정. **두 출처 모두 Laozi를 탈출 성공자로 인정한다는 점은 일치.**

### 3. 옥황상제/God의 현재 상태

| 출처 | 주장 |
|------|------|
| **Yeogeu** | God (옥황상제)가 AD 1127년 Domain에 의해 파괴됨. 죽었음 |
| **Alien Interview** | Old Empire 본부/중앙통제센터는 아직 발견되지 않음. force screen 아직 작동 중 |

**분석**: Yeogeu는 God이 "죽었다"고 하지만, Alien Interview는 시스템이 "아직 작동 중"이라고 말함. 창조주는 죽었어도 시스템이 자동으로 돌아가는 상황일 수 있음.

**잠정 결론**: God의 개인적 존재는 소멸했으나, 그가 설치한 prison system (force screen, amnesia stations, 최면 기계)은 자동 작동 중. 이것이 말법시(상법 시대) — God의 법이 그림자로만 남아 약해졌지만 여전히 돌아가는 상태 — 라는 Yeogeu의 설명과 정확히 일치.

## 불일치의 원인에 대한 가설

이 3가지 불일치는 **각 출처가 바라보는 관점의 차이**에서 비롯된 것으로 보인다:

1. **Alien Interview**: Domain의 외부 관찰자 시점. 전체 그림을 보지만, 지구 내부의 미묘한 차이를 놓칠 수 있음
2. **Yeogeu**: 450억 IS-BE 중 하나가 내부에서 싸워 얻은 직접 경험. 디테일은 풍부하지만, 전체 우주적 맥락은 제한적
3. **Montalk**: 에너지/진동/밀도 기반 분석. spiritless humans는 영적 기능 완전 상실 상태를 설명하는 기술적 용어
4. **Blog**: 개인적 깨달음 + 각성 과정. 자신의 경험을 NPC/Player 프레임워크로 표현

**진리는 이 모든 관점을 종합할 때 가장 선명해진다.**
